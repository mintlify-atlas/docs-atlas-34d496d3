## What is JS Beautify?

JS Beautify reformats and re-indents JavaScript, CSS, and HTML source code. It handles minified code, unpacks scripts packed by Dean Edward's packer, and partly deobfuscates scripts processed by `javascript-obfuscator`.

You can use it in multiple environments:

## Key features

<AccordionGroup>
  <Accordion title="JavaScript, CSS, and HTML beautification">
    Reformat minified or poorly indented source code back to a human-readable form. Supports ES6+, JSX, and modern CSS.
  </Accordion>
  <Accordion title="40+ configuration options">
    Control indentation style, brace placement, line wrapping, newline behavior, operator position, and much more â€” all from a single config file or CLI flags.
  </Accordion>
  <Accordion title="EditorConfig integration">
    Automatically inherit indentation settings from your project's `.editorconfig` file by passing `--editorconfig`.
  </Accordion>
  <Accordion title="Templating language support">
    Beautify HTML templates written with Django, Handlebars, ERB, PHP, Smarty, and Angular control flow syntax without corrupting template tags.
  </Accordion>
  <Accordion title="Beautify directives">
    Embed `/* beautify ignore:start */` / `/* beautify ignore:end */` or `preserve` directives directly in your source to protect sections from reformatting.
  </Accordion>
  <Accordion title="Language-specific overrides">
    Apply different indent sizes or settings to JS, CSS, and HTML through a single `.jsbeautifyrc` config file with nested language keys.
  </Accordion>
</AccordionGroup>

## Quick example

<CodeGroup>

```bash npm (global CLI)
npm install -g js-beautify
js-beautify foo.js
```

```bash pip
pip install jsbeautifier
js-beautify foo.js
```

</CodeGroup>

```javascript Node.js library
var beautify = require('js-beautify');

var ugly = 'function foo(x){if(x>0){return x;}else{return -x;}}';
console.log(beautify.js(ugly, { indent_size: 2 }));
// Output:
// function foo(x) {
//   if (x > 0) {
//     return x;
//   } else {
//     return -x;
//   }
// }
```
